# Aqui Perto

Focus timer and study analytics - sessions, streaks, and subject-level progress tracking. Monorepo with **NestJS** backend and **Angular 22** frontend.

## Tech Stack

| Layer        | Technology                                                    |
| ------------ | ------------------------------------------------------------- |
| **Frontend** | Angular 22, Signals, TailwindCSS 4, Spartan UI, ApexCharts    |
| **Backend**  | NestJS 11, Passport JWT, Prisma 7, class-validator            |
| **Database** | PostgreSQL (Neon serverless)                                  |
| **Tooling**  | npm workspaces, Biome, Vitest, Jest, Playwright, Concurrently |
